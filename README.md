# 🦜 オカメの気持ち — Okame no Kimochi

## 概要
「オカメの気持ち」は、オカメインコの体重記録をもとに  
さし餌回数の切り替えタイミングを自動で提案する  
**Excel＋Python連携システム**です。

Excel上で日々の体重を入力するだけで、  
備考欄に「次の給餌ステップ」や「警告」が自動で表示されます。  
また、制作意図・構成・処理の流れをまとめた発表資料（.pptx）も付属しています。

---

└─ README.md # この説明書

## 🧭 操作手順

### 1. Excelファイルの準備
1. `オカメの気持ち.xlsm` を開く  
2. シート上の **左上赤枠部** に初期値を入力  
　　- 例：生年月日、下限体重など  

---

### 2. 日付の自動生成
1. **右上部の「状況確認」ボタン** をクリック  
2. 対象欄に日付が自動生成される  

---

### 3. 毎日の体重入力
1. シート上の **B列（朝体重欄）** に毎朝の体重を入力  
2. 対象欄に自動で日付が反映される  

---

### 4. 備考欄の自動判定
- 手順3を続けていくと、**備考欄に給餌指示が自動で表示**されます。  
  - 例：「明日から2回給餌に切り替え」「下限体重注意」など  
- この自動判定は `okame.py` がExcel内データを解析して実行しています。  

---

## 🧠 Pythonスクリプトの仕組み
- `win32com.client` を使用してPythonからExcelを直接操作  
- 生後日数・平均・3日／5日移動平均を算出  
- 下限体重割れ時は赤セルで警告  
- 条件に応じて「3→2→1→0回」への給餌切り替えを自動判断  

---
